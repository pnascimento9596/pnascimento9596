## Flagship Project

### [CEIntel — Clinical Engineering Intelligence](https://github.com/pnascimento9596/ceintel)

A Python-based RFP evaluation engine for healthcare equipment service procurement ($15–25M contracts). Normalizes structurally incomparable vendor bids, detects hidden costs using 8 pattern-based rules, and scores vendors on weighted clinical engineering criteria.

**Why it exists:** Health systems routinely compare a $17M pass-through bid against a $20M full-service bid without decomposing OEM markup, ghost asset inflation, or coverage level mismatches. CEIntel solves this.

`103 tests` · `5 CLI commands` · `8-tab Excel reporting` · `--demo mode`

```bash
git clone https://github.com/pnascimento9596/ceintel.git && cd ceintel
pip install -e ".[dev]" && ceintel rfp compare --demo
```
